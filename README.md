# How To Run

##  In a terminal

### Run MongoDB

```
docker run --name testemongo -d -p 27017:27017 -p 28017:28017 -e AUTH=no mongo 
```

### Run App

```
STORM_GLASS_API_TOKEN=<PUT_HERE_YOUR_API_TOKEN> yarn start
```
